Accessible First

Skip link, clear landmarks, proper headings, focus states, and high-contrast defaults.

What else is accessible here?

Color contrast, logical tab order, link focus rings, form labels, and ARIA where helpful.

FAQ

How do I customize colors?

Edit CSS variables in :root inside styles.css.
